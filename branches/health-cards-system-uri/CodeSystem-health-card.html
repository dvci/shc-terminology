CodeSystem: Health Card Type Code System

Official URL: https://smarthealth.cards# Version: 0.1.0
Active as of 2022-09-11 Responsible: VCI Computable Name: HealthCardTypeCodeSystem

A code system defining various credential types for SMART Health Cards

Implementation note: In a SMART Health Card, values of the vc.type element SHALL be in the format https://smarthealth.cards#$SOME_CODE, where $SOME_CODE is replaced by one of the codes listed below.

For example, all SMART Health Cards have the value https://smarthealth.cards#health-card in vc.type array. Additional values may also be included in this array. For more information, please see the SMART Health Card specification.

Code Description(s)
health-card A VC designed to convey a "Health Card" (i.e., clinical data bound to a subject's identity)
covid19 A Health Card designed to convey COVID-19 details
monkeypox A Health Card designed to convey monkeypox details
immunization A Health Card designed to convey immunization details
laboratory A Health Card designed to convey laboratory results
