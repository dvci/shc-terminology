Terminology: Value Sets

These define sets of codes used by systems conforming to this implementation guide.

Immunization / All / CVX

Contains all CVX codes from <https://www2.cdc.gov/vaccines/iis/iisstandards/vaccines.asp?rpt=cvx> as of 2023-06-23.
Immunization / All / ICD-11

Contains ICD-11 MMS codes from the 2023-01 release that are descendants of the Foundation element 164949870 (“Vaccines”) according to the ICD API. Generated on 2023-06-23.
Immunization / All / SNOMED CT

Contains descendants of SNOMED CT concept 787859002 “Vaccine product (medicinal product)”. Note that codes from any SNOMED CT extension are valid as long as they descend from this concept.
Immunization / COVID / All codes

A list of all immunization codes for COVID, combining the following ValueSets: <a href=”ValueSet-immunization-covid-cvx.html”>ValueSet-immunization-covid-cvx.json</a>; <a href=”ValueSet-immunization-covid-snomed.html”>ValueSet-immunization-covid-snomed.json</a>; <a href=”ValueSet-immunization-covid-icd11.html”>ValueSet-immunization-covid-icd11.json</a>
Immunization / COVID / CVX

Contains CVX (“vaccine administered”) codes with recommended consumer-suitable display text for use in SMART Health Cards, with localized consumer friendly terms
Immunization / COVID / ICD-11

Contains ICD-11 MMS codes from the 2023-01 release that are descendants of XM68M6 (“COVID-19 Vaccines”).
Immunization / COVID / SNOMED CT

Contains SNOMED CT codes that identify specific COVID-19 vaccines for use in SMART Health Cards.
Immunization / orthopoxvirus / All codes

A list of all immunization codes for orthopoxvirus, combining the following ValueSets: <a href=”ValueSet-immunization-orthopoxvirus-cvx.html”>ValueSet-immunization-orthopoxvirus-cvx.json</a>
Immunization / orthopoxvirus / CVX

Contains CVX (“vaccine administered”) codes with recommended consumer-suitable displays for use in SMART Health Cards, with localized consumer friendly terms
LOINCs identifying qualitative COVID lab tests

This value set includes a subset of the LOINCs found at <https://loinc.org/sars-cov-2-and-covid-19/> that identify COVID-19-related laboratory tests. Only those laboratory tests that include qualitative results we believe to be useful in SMART Health Cards are included.
Qualitative infectious disease-related lab test result codes

This value set includes a codes to identify the results of qualitative lab tests related to infectious diseases.

Terminology: Code Systems

These define new code systems used by systems conforming to this implementation guide.

Designation Use

Code system indicating how a display name should be used in our community-maintained localizations. This is NOT used in SMART Health Cards themselves – it is just for the community-provided localizations ValueSets for displaying SMART Health Card codes in user interfaces.
Health Card Type Code System

A code system to identify a SMART Health Card. Previously this code system was used to differentiate between different types of SMART Health Cards, but this has been deprecated. See the SMART Health Cards spec for details.
Identity Assurance Level Code System

Code representing identity assurance level, based on NIST 800-63-3

Terminology: Concept Maps

These define transformations to convert between codes by systems conforming with this implementation guide.

Mapping between CVX and AIR Covid Codes

A Mapping between CVX and AIR Codes for Covid vaccinations. This mapping is bi-directional - one group for each direction
