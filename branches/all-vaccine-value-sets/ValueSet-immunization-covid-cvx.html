Contains CVX (“vaccine administered”) codes with recommended consumer-suitable display text for use in SMART Health Cards, with localized consumer friendly terms

References

This value set is not used here; it may be used elsewhere (e.g. specifications and/or implementations that use this content)

Source data

This value set contains all CVX codes as published by CDC related to COVID-19.

Updating this value set

The CVX codes from CDC are curated into a JSON file on GitHub:

Edit on GitHub

A helper script can be run to automatically add newly published CVX codes. This script will preserve manually defined consumer-friendly display text in this JSON file.

If you manually edit this JSON file, please run prettier to ensure your JSON is valid and the formatting stays consistent. Please see the README on GitHub for more information.
